# Quiz_Application_
GeoQuiz - Android Quiz Application
The app presents geography-based questions to users and provides immediate feedback through interactive UI elements.

----------------------------------------------------
Key Features

Question Interface: Displays geography questions using a TextView.

Interactive Buttons: Includes "True" and "False" buttons for user input.

Toast Feedback: Triggers short, non-intrusive pop-up messages (Toasts) to inform users if their answer was correct or incorrect.

Resource Management: Centralizes all UI strings in a dedicated strings.xml file for better maintainability.
